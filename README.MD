## 参数
 - -env api_key: 你的api key
 - gui: 启动gui(默认)
 - nogui: 不启动gui
 - -env base-url: 自定义api base url(默认***https://api.deepseek.com/anthropic***)
 - -env model: 自定义模型(默认***deepseek-v4-flash***)
 - -env system_prompt: 自定义系统提示(默认***你是一个有用的AI助手。***)
 - -env max_tokens: 自定义最大token数(默认***1000***)
 ### -env 仅为创建文件请自行修改env文件
